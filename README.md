Headphone Landing Page

A premium, futuristic dark-mode landing page for headphones. Designed to showcase features, pricing, and product highlights while guiding users toward purchasing.

🎯 Goal

Create a clean, responsive landing page with strong visual hierarchy, engaging interactions, and clear CTAs.

✨ Features
Hero Section: Headline, subtext, glowing CTA, premium mockup
Features Section: 3–4 cards (Noise Cancelling, Deep Bass, Wireless, Long Battery)
Gallery Section: High-quality product images, hover zoom effects
Pricing Section: 2–3 plans with highlighted option and CTA
Testimonials Section: User reviews with avatars
Product Highlight & Final CTA Sections: Split layout with image, text, and strong CTA
Footer: Minimal dark footer with links and social icons
🎨 Design System
Dark mode UI, glassmorphism cards, neon blue accents
Typography: Poppins with Major Third scale
8pt grid & spacing system
Components: Buttons, cards, navbar, footer built with Auto Layout
Smooth hover effects and micro-interactions
🛠 Tools
Designed in Figma
Prototype CTA flow (Hero → Product → Checkout)
Assets ready for developer handoff
